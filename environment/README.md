# Environment Setup

For Linux you can reference the files below for detailed information or run the automated [SetupLinux.sh](./SetupLinux.sh) script in this directory.

- [Environment](./Environment.md) — core packages, package managers, Git, and dotfiles.
- [Terminal](./Terminal.md) — shell, prompt, SSH, and terminal setup.
- [Coding Application Configurations](./Coding-App-Configs.md) — editor and tool settings.
- [Local AI Model Setup for Zed](./Local-AI-Model-Setup.md) — run an Apache-2.0 Qwen model locally with `llama.cpp` and connect it to Zed's AI agent.
- [Editor Extensions](./Editor-Extensions.txt) — recommended editor extensions list.
