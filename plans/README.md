# Setup Plans

Each file in this directory is the ordered human-facing route for one entry point. Plans link reusable catalog resources and may include manual, automated, and hybrid work in the required order.

Provisioning adapters execute only automated steps. The plan remains the authoritative route for a human completing the setup.

## Available plans

- [Linux](./linux.md) — automated base provisioning plus manual configuration.
- [macOS](./osx.md) — manual application and development-environment setup.
- [Android](./android.md) — manual source and privacy decisions.
- [Windows](./windows.md) — manual application and device setup.
- [Synology](./synology.md) — manual NAS and container configuration.
- [Printers and scanners](./printers.md) — device-specific driver setup.
- [Self-hosting](./self-hosting.md) — workload selection and verification.
